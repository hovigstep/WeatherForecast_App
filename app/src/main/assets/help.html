Help Page

Welcome to the Help Manual for WeatherForecast App.

This is App gathers Weather Information from Openweather Api's and displays the Daily and weekly Forecast in Fragments. It requires the following Permissions : Internet, Access Network State, Fine Location and Coarse location. This app does not have access to Internal storage.

1. To change City for which Forecaset is viewd, please go to settings

2. To Add Bookmarks Please use Text Edit to enter.

3. To Remove Bookmark Long Click Bookmark.

4. To Move to Map from Bookmark Short Click Bookmark.

5. To Place a pin on map please enter location.
